My job was to create from scratch task manager app. My biggest impact was in
implementing workflow to create auth, organisations, teams, invite or remove people;
customisation of user profile, organisation or team info; making pages responsive and
accessible, etc. The crucial thing here was to take care about UI and UX, so I made a
project easy to maintain and expand, while keeping its performance high.

I was also responsible for creating and maintaining CI/CD pipeline, making sure the
project is always up-to-date and deployed to production.

Main gimmick was each task ticket is associated with some kind of reward. Money, time off, vacation, or special gift. It was a great challenge to implement this feature, but it was worth it. It was a great motivation for users to complete their tasks.

#### Stack: React 18, TypeScript, Redux, react hooks, react router, date.js, axios, Cookies, SASS, ESLint, TSLint, Vercel, Stripe, Plaid, Figma
