My main goal was to improve all gadgets. Fixing bugs, refactoring code, and implementing new features. Added more options for customization of shown data, new data fields and filters for users to fill, improving UI in order to reduce the amount of clicks needed to reach the goal, and improving errors handling.

Gadgets was targeted on analysis of users data, how productive they are and how many tasks were in done in what time. Lots of option how exactly you wish to see your data

#### Tech Stack: atlaskit, clsx, React 16, Redux, VictoryCharts, babel, TypeScript, enzyme, mocha, Sass, React Hooks
