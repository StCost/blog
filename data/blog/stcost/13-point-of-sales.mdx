Point of Sale is a project made for restaurant owners to help them manage their business. Application is installed on device (usually tablet) handling the orders and transactions. Staff can easily create new orders, modify items, process payments, and view order history. Customers can be looked up by their phone number or name, or they can manually make order on separate device and UI.

I implemented UI components, unit tests, logic for ordering, refactored old code, added documentation for team members, polished the application, and fixed bugs. Added custom orders. Took part in improving most parts of application.

The application features various routes for different functionalities such as new order creation, item modification, payment processing, order history viewing, and customer lookup. The application also supports different station types for order creation, catering to different business models.

#### Tech Stack: React, TypeScript, Sass; MaterialUI, Redux, Sentry, axios, lodash, [Create React App](https://github.com/facebook/create-react-app)
