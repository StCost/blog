# Implemented COLLAPSE MACHINE Game Features

- ! Infinite* world generation. *(40mil km)
- Day-night cycle
- Player Character Controller (walk, sprint, crouch, coyote time, step up (climb up))
- ! NPC agents (patrol, idle, faction (enemy or ally), flee, chase, die, ragdoll, dismemberment, inventory, melee attack, gun attack, grenade, heal)
- Player controls NPC agent, so behaves in similar way as NPCs
- ! Voxel terrain, allowing add/remove terrain and mix materials
- ! Persistent world (entities, health, position, opened doors, destroyed objects, dead enemies, terrain operations, etc)
- ! Networked co-op multiplayer (no players limit)
- Player inventory with 12 slots (for each alpha numberic button: 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, -, =)
- ! Boxes to store other items physically (throw item inside box, close box, take box with you). No UI!
- Interactable items to take (E) and to throw (Q) or put down (hold Q), or rotate (hold Q + Scroll)
- Items are shown holstered on body, if small enough
- Holstered glowing items (lamp) illuminates environment
- Level Of Details (LODs)
- First Person View and Third Person View (toggle on F5)
- Drivable heavy cargo vehicle with walkable interior
- ! Closing interior if vehicle allows to drive it, while physical objects freely placed in interior. Physics of outside world does not affect items inside. Allowing *unique* interactive vehicle interior while someone else drives the vehicle in open world
- Weather, randomized or localized (Wind, Rain, Thunder, Ligthning)
